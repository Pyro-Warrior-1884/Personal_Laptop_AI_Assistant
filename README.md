# Personal Laptop AI Assistant
## Started this project as simple passion project for creating a personalised ai assistant (named BMO) for my laptop
## Inspiration started off from seeing the ai helper jarvis from the marvel cinematic universe
## The project is a far cry from completion but a work in progress nonetheless

## Features

- local LLM downloaded via ollama (Model: gemma:2b)
- Voice input using Vosk speech recognition
- Text-to-speech via ElevenLabs API
- Access to Chrome
- Cross-platform support (Windows/Linux)
